# Mastercard enables tokenized agent transactions

Mastercard said all U.S. banks' cards were capable of conducting an agentic transaction by the end of the prior year. It described Agent Pay as providing agents with network tokens, identity, transparency, issuer visibility, fraud, safety, and security controls.

## Why it matters

This is a broad enablement milestone and a concrete control architecture. It is readiness evidence, not proof that every enabled card has generated agent volume.
