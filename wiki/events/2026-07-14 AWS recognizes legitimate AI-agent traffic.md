# AWS recognizes legitimate AI-agent traffic

AWS published a production control pattern for authenticating legitimate AI-agent traffic with AWS WAF Bot Control.

## Why it matters

Agent traffic becomes economically addressable only when it can be distinguished from unidentified automation, governed by policy, and measured separately.
