# Cloudflare presents Web Bot Auth for autonomous commerce

Cloudflare presented Web Bot Auth as a way for merchants to cryptographically verify agent identity before processing transactions and bind that identity to specific transaction flows for fine-grained authorization. The cited slide named American Express, Mastercard, and Visa as payment-network adopters.

## Why it matters

This is announcement and readiness evidence that identity and authorization are being positioned as part of the transaction path for autonomous commerce. The slide does not establish live merchant traffic, transaction volume, or independent revenue.
