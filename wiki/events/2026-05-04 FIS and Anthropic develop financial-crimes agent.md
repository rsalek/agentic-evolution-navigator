# FIS and Anthropic develop financial-crimes agent

FIS and Anthropic announced a Financial Crimes AI Agent for anti-money-laundering investigations. BMO and Amalgamated Bank were in development with the agent, while broader availability to FIS financial-institution clients was planned for the second half of 2026.

## Why it matters

This is named-bank pilot evidence for an agent that operates inside a system-of-record provider's governed infrastructure. FIS reports a target of reducing investigation work from hours or days to minutes, but it provides no deployed production metric, so the claimed outcome remains prospective.
