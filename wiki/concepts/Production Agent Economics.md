# Production Agent Economics

The end-to-end cost and value of completed agent work after latency, failures, supervision, security, and integration are included.

## Observable signals

- Cost per successful task, not token cost alone.
- Completion rate, latency, exception rate, and human-review time.
- Revenue per task, resolution, transaction, or attributed conversion.
